# R14 — State machine non-atomic

Naive switch order: re-index RDF → persist settings. If interrupted between steps — settings file shows old active profile, but RDF graph filtered к new profile. UI confused: «current = X» but SPARQL/search returns Y.

Mitigation: B.4 journaled two-phase commit reverses order. Settings persisted BEFORE filesystem. Journal records intent. Reload recovery routine handles incomplete state.
