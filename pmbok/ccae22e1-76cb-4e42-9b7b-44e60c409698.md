# R9 — Path traversal via AssetSpace_git URL

Если AssetSpace_git property contains malicious URL (`https://attacker.com/repo` или `https://github.com/foo/../bar/etc/passwd`) — plugin может download to unexpected path или leak credentials to wrong host.

Mitigation: strict regex allowlist enforces github.com domain + clean owner/repo format. Folder name sanitization prevents path-component manipulation.
