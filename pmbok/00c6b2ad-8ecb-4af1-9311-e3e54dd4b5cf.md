# R8 — PAT leaked в plugin error logs

GitHub REST 4xx/5xx responses может include the Authorization header reflected back в error body. If plugin logs raw error без sanitization → PAT в console / log files / bug reports.

Mitigation: B.1 redaction layer is mandatory wrapper around all error message constructions. Pattern matches GitHub PAT formats (ghp_, gho_, ghu_, ghs_) → replaces with ***REDACTED***. Unit-test enforced.
