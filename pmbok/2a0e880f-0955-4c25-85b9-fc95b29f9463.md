# R10 — Zip-slip tarball attack

Malicious tarball entry с path `../../etc/passwd` или `/etc/cron.daily/evil.sh` → if extractor naively writes к the path, gets arbitrary filesystem write outside staging dir. Classic CVE-2018-1002101 pattern.

Mitigation: B.2 TarExtractor strict path validation pre-write. Adversarial test fixtures encoded в test suite.
