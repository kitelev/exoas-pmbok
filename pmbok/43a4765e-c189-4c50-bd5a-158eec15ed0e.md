# R7 — PAT scope over-broad

Classic GitHub PAT (no fine-grained restrictions) grants access to ALL user repos. If plugin or any downstream consumer compromised, attacker gets full GitHub identity.

Mitigation: documentation insists fine-grained PAT с per-repo allowlist (`exoas-*` pattern). B.8 Settings «Test connection» surfaces actual scope so user verifies безопасность setup.
