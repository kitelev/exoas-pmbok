# R6 — PAT-in-Sync risk

Если PAT stored в обычном plugin `data.json`, Obsidian Sync уплывает credentials к каждому device + cloud storage. Catastrophic confidentiality breach + PAT requires manual rotation.

Mitigation: separate `data.local.json` для secrets. Plugin loads dual settings (synced data.json + local data.local.json). PAT lives only в device-local file.
