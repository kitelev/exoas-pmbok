## 2. Business Case

Use case [[d8725272-4b4d-4e68-bf01-ce9b111bbe12]] требует, чтобы опытный пользователь добавлял кнопку «Create Next Iteration» на ассеты класса `ems__Task` за <60 секунд, **без правки TS-кода**. Это acceptance gate для milestone «Editable Homoiconic UI alpha» из [[4055cfe6-648a-46ef-b437-c2cf18611ce1]] (Pillar Editable Homoiconic UI).

Сейчас существующий `executeCreateInstance` (RFC-016) поддерживает создание ассетов с targetClass/Prototype/Folder, но **не копирует frontmatter из `$target`-ассета** и пишет hardcoded `exo__Asset_source` back-link. Это блокирует use case.

Spike (2026-05-13) показал: RFC v0.1 reinventing existing RFC-016 vocabulary (3 из 5 proposed properties). v0.2 — minimal scope: extend, не parallel.

## 3. Цели (SMART)

- **Specific:** Расширить `executeCreateInstance` двумя возможностями: copy-from-target (blacklist-default) + configurable back-link через `exocmd__Grounding_linkBackProperty` (fallback на `exo__Asset_source`)
- **Measurable:** `time_to_first_working_button` median ≤60s (n=5, опытный пользователь)
- **Achievable:** 7-10h budget per Migration Engineer estimate (post-spike); extend 1 existing case, не новый dispatch
- **Relevant:** Прямой acceptance gate для Strategy 2026 H1 Pillar Editable Homoiconic UI
- **Time-bound:** M4 (release) — 2026-05-18
