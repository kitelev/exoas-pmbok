profile-personal в vault-2025 vs vault-exodev может иметь разный set применимых ontologies (vault-2025 has $kitelev*; vault-exodev has $exodev*). Single profile asset не fit оба vaults.

Probability + Impact Low: addressed через vault-local profile creation (different UID per vault). User decides if cross-vault sync needed — separate concern.
